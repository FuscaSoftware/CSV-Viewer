# Fusca CSV-Viewer
A small PHP Scripset to view csv-Files in Datagrip-Tables


## Requirements

* PHP-Environment e.g. Xamp/Mamp or Webspace (For Help, please contact Fusca.de)
* PHP Version 7.1+
* a CSV-File
* Browser
